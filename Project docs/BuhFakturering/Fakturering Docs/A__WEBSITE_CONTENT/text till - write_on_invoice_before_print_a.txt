Du kan alltså skriva valfri text på fakturan innan du skriver ut och det kommer
finnas med på utskriften. 

Det kan vara nyttigt vid många tillfällen. Har du pågående kampanj kan 
du bara skriva några rader direkt på fakturan för att informera dina kunder.

Du kan också ändra eller radera text på alla ställen i fakturan innan utskriften. Har
du stavat fel? Ändra direkt på fakturan!
